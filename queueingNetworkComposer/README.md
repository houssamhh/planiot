# queueingNetworkComposer

To run the queueing network composer, create a new project in Eclipse. Add the following jar files as project dependencies:

* json-20210307

* opencsv-3.8

* JMT-1.2.0

* JMT-singlejar-1.2.0

Update the input and output file paths at lines 90 and 91. Run the class iotSys.broker.App as a Java Application. The JMT XML file that represents the performance of data flows will be generated (variable OUTPUT_FILE_NAME at line 91).
